## 中文说明
  
### 简介

Mysplash是一款轻量、简洁的图片（壁纸）软件，资源来自无版权图片网站[Unsplash.com](https://unsplash.com/)。


网站内有1W+的图片总数，每10小时刷新10张摄影图片，每10天推出10张优秀作品，由网友自行上传分享，可用于商用。
